# Hermes Agent Kit

> AI Agent Development Toolkit — from **Dia Agency**

Build, deploy, and orchestrate AI agents with Hermes. This is the development kit for creating multi-agent systems, MCP servers, and autonomous AI workflows.

## Features

- **Multi-Agent Orchestration** — Coordinate teams of specialized AI agents
- **MCP Integration** — Model Context Protocol servers and tools
- **Harness Engineering** — Design toolsets, capabilities, and profiles
- **DevTeam Studio** — 6 specialized sub-agents (AETHER, VULCAN, KAI, SAINT, PHOENIX, AURA)
- **GitHub Integration** — PR management, code review, CI/CD

## Structure

```
hermes-agent-kit/
├── agent-registry/       # Agent profiles and configurations
├── docs/                 # Documentation
├── skills/               # Reusable skill definitions
├── templates/            # Agent and tool templates
└── scripts/              # Automation scripts
```

## Getting Started

1. Clone this repo
2. Review the documentation in `/docs`
3. Start building your agent team
