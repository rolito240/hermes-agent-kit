# Setup Guide

## Prerequisites
- Hermes Agent v0.14+
- Python 3.10+
- GitHub CLI (optional)

## Configuration

1. Clone this repository
2. Set up environment variables:
   ```bash
   export GITHUB_TOKEN=your_token_here
   ```
3. Configure Hermes to use the agent profiles

## First Run

```bash
hermes agent list
hermes agent run <agent-name>
```
